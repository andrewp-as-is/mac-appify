### Installation
```bash
$ [sudo] pip install mac-appify
```

```bash
$ [sudo] npm i -g mac-appify
```

#### Features
+  script [shebang](https://en.wikipedia.org/wiki/Shebang_(Unix)) required

#### Examples
```bash
$ appify script.sh name.app
$ appify script.sh name.app Icon.png
```
